# Movies Game

Application allow users to search movies and check the details of each movie

### Demo
https://movies-app-phi-jade.vercel.app/

### Installation

```sh
$ cd movies-app
$ npm install
```
### Runing the app
Starting react app at port 3000
```sh
$ npm start
```

### Packages
-React
-Sass

### API used to seach movies
https://www.omdbapi.com/
